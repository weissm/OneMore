
Set-RainbowSections.ps1
Colorize all sections in the current notebook in rainbow color sequence.

This is an example of embedded CSharp in a PowerShell script to make it easier
to navigate and manipulate XML.

--------------------------------------------------------------------------------

0. Save Set-RainbowSections.ps1 to your local drive in some <path>
1. From the OneMore More menu, choose Run Plugin
2. Enter a name if you want to save this as a favorite such as "Set-RainbowSections"
3. Enter the command: powershell
4. Enter arguments: -file <path>\Set-RainbowSections.ps1
5. Set the Target field to 'Current notebook'
6. Click Save to save on the File/Plugins menu
7. Click Run to run now or activate from the File/Plugins menu
